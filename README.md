2048 is a single-player sliding tile puzzle video game written by Italian web developer Gabriele Cirull. The objective of the game is to slide numbered tiles on a grid to combine them to create a tile with the number 2048. Here is a Python version that uses TKinter
![2048_game](https://github.com/MeghashreeP-6/2048-Game-Learning/assets/46756957/eb095b17-f43a-460e-bc15-cff90d1a338f)
![2048_Functions](https://github.com/MeghashreeP-6/2048-Game-Learning/assets/46756957/e3e18d68-2266-454b-80ad-73d52c2a8767)

To start the game run:
$ python3 puzzle.py
